# FFmpeg Binaries

Bu papkaga FFmpeg binary fayllarini joylashtiring.

## Yuklab olish:

1. **Rasmiy sayt:** https://ffmpeg.org/download.html#build-windows
2. **To'g'ridan-to'g'ri havola:** https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip

## Joylashish:

Zip faylini ochib, `bin` papkasidagi fayllarni shu yerga ko'chiring:

```
ffmpeg/
├── ffmpeg.exe
├── ffprobe.exe
└── ffplay.exe
```

## Avtomatik yuklab olish:

```bash
python download_ffmpeg.py
```
